# Bitcoin Market Sentiment vs. Trader Behavior Analysis

This project investigates the relationship between the **Bitcoin Fear & Greed Index** and trader performance on the **Hyperliquid** platform. By analyzing over 211,000 trade records, we explore how market sentiment drives profitability, risk-taking, and directionality.
